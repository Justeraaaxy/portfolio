A modern, responsive portfolio website showcasing my work as a self-taught web developer with 3+ years of experience.

## 🚀 Features

- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop
- **Modern UI**: Clean, professional design with smooth animations
- **Multiple Sections**:
  - Hero section with call-to-action buttons
  - About me with statistics
  - Work experience timeline
  - Featured projects showcase
  - Technical skills categories
  - Contact information
- **Smooth Scrolling**: Navigation with smooth scroll behavior
- **Mobile Optimized**: Hamburger menu for mobile devices

## 🚀 Getting Started

1. Clone the repository
```bash
git clone https://github.com/Justeraaaxy/portfolio.git
cd portfolio
```

2. Open `index.html` in your browser
```bash
open index.html
```

## 📦 Deploy to GitHub Pages

1. Go to repository settings
2. Scroll to "GitHub Pages" section
3. Select `main` branch as source
4. Your site will be live at `https://Justeraaaxy.github.io/portfolio`
